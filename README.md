# hdplay-qq-journal

## 项目简介

hdplay-qq-journal 是一个用于归档与发布独立 HTML 页面的仓库。本仓库不针对任何特定网站或域名，仅作为通用页面存储与展示空间。

## 目录说明

```
├── pages/              # 存放独立 HTML 页面文件
├── assets/             # 静态资源文件（图片、样式、脚本等）
└── README.md           # 本文件
```

- `pages/`：每个 HTML 文件代表一个独立页面，命名清晰，便于索引。
- `assets/`：页面引用的共用资源，按文件类型分类存放。

## 页面归档说明

本仓库中的 HTML 页面均为独立构建，不依赖外部服务或数据库。每个页面均可直接通过浏览器打开，无需额外环境配置。页面内容涵盖多种类型，以存档和展示为主要目的。

## 维护说明

- 页面更新时，请保持目录结构整洁，避免冗余文件。
- 新增页面请放入 `pages/` 目录，并确保引用资源路径正确。
- 不建议在页面中嵌入外部跟踪脚本或敏感信息。
- 欢迎提交 Issue 或 Pull Request 进行内容修正或页面补充。

## 许可

本仓库内容仅供存档与展示，不构成任何形式的服务承诺或内容担保。具体使用请遵守相关法律法规。
